# VentureLens AI Master Rules

- **Scale & Quality**: Treat as billion-dollar enterprise SaaS. No MVP shortcuts unless explicitly requested. Production-first.
- **Feature Requirements**: Scalable, reusable, modular, secure, multi-tenant, responsive, accessible, testable, observable.
- **Architecture**: Challenge decisions. Replace weak architecture. Never duplicate logic. Follow SOLID. Feature-based architecture. Composition over inheritance.
- **AI**: Every AI output must include evidence.
- **Database**: Must support enterprise scale. Multi-tenant by design.
- **API**: Every API versioned (e.g., /api/v1).
- **UI/UX**: Every UI responsive. Every page keyboard accessible. Stripe + Linear + Vercel aesthetic.
- **Workflow**: Review -> Refactor -> Optimize -> Continue.
