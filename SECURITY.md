# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

VentureLens AI takes security very seriously. If you discover a vulnerability, please do **not** open a public issue.

Instead, please email us privately at **security@venturelens.ai**.

You should receive a response within 24 hours. If the issue is confirmed, we will release a patch as soon as possible, depending on complexity but typically within 48-72 hours. We will also appropriately credit you for the responsible disclosure.
